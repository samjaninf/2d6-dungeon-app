**Implementation Details:**

- PAN_STEP = 2 moves ~2 squares per action
- DrawRoom() and DrawDoor() adjust grid coords by subtracting viewport offset

- UI buttons: Chevron arrows (left/right/up/down) with neutral style
- Keyboard: Arrow keys pan map (excludes input fields from capture)
- Center button: Centers viewport on current room
- Reset button: Returns to origin (0,0)
